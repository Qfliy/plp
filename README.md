Programing language plp
=========================

**Plp** is a *programming language* that contains basic functions

version: alfa-2.5

What's new
-------

- priorety for ```()``` new operators
- the code became cleaner

Syntax
-----------

```erl
% comment
foo: 3;
test: 10 - (4 + foo);

puts test; % output 3

function {
  var_x: 4;
  puts var_x; % 4
}

!function; % call function

puts var_x; % 4

```

Run Program
-----------

```ps1
plp "path/to/file.plp"
```
